# Project Brief

**Last Updated**: 2024-01-15

## Overview

AI-powered demand letter generation tool for legal professionals. Built with Next.js 14+, TypeScript, PostgreSQL (Prisma), AWS S3, and OpenRouter API.

## Core Value

Generate professional demand letters using AI with style/tone matching, medical record processing, and document analysis.

## Tech Stack

- **Framework**: Next.js 14+ (App Router), React 18, TypeScript
- **Database**: PostgreSQL + Prisma ORM
- **Storage**: AWS S3
- **AI**: OpenRouter API (Claude 3.5 Sonnet, GPT-4o)
- **Auth**: JWT
- **Deployment**: Render/Vercel

## Key Features

- Document management with version control
- AI-powered section generation
- Template system (Standard, PI, UIM, 3P Liability)
- Medical data extraction and chronology
- Media transcription
- Style/tone matching from uploaded documents
- Word/PDF export
